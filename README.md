# Darkness Of Control
Um jogo inspirado e baseado em Dead By Daylight e Mogeko Castle, tem objetivo de trazer a estética antiga dos jogos (Slenderman The Eight Pages, Slendytubbes) para uma nova geração.

> Mêcanicas
Explore, repare, fuja, e trabalho em equipe!
Repare os 8 geradores para abrir as portas da liberdade, colete chaves para coletar ferramentas com função de reparar mais rápido!
Cada assasino tem suas habilidades e forças. E vale também para sobreviventes.


> Aparência
Estética retrô com fusão moderna de jogo VHS
Leve e rápido para os PC/Notebooks que não rodam nem o Free Fire!
Arte em pixel para a estética retrô.
